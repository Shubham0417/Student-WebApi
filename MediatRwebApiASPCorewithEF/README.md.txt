## How to Run

1. Clone repository

git clone https://github.com/Shubham0417/student-api.git

2. Navigate to project

cd StudentService.API

3. Run project

dotnet run

4. Open Swagger

https://localhost:xxxx/swagger

Database will be created automatically.
